# Utilización básica de Git, GitHub y Markdown #
## Clonar repositorio ##

Comandos:

git clone https://github.com/japegon/campusciff.git

## Commit inicial ##

Comandos:

git add README.md

git commit -m "commit inicial"

## Push inicial ##

Comandos:

git push -u origin master

## Ignorar archivos ##

Comandos:

gedit .gitignore
	$GIT_DIR/privada
	/privado.txt

git add -A

git commit -m "ficheros privados"

git push -u origin master

## Crear etiqueta ##

Comandos:

git tag v0.1

git commit -m "etiqueta"

git push -u origin master
